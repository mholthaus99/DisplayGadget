Handler for IR digit button presses. 

If the current mode is MODE\+\_\+\+FUNC, performs an action based on the digit\+:
\begin{DoxyItemize}
\item 1\+: Toggle backlight and return to default mode.
\item 2\+: Switch to device info mode.
\item 3\+: Switch to climate mode.
\item 4\+: Return to default mode. else does nothing
\end{DoxyItemize}
